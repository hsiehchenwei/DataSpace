API 金鑰設定指引

1

前往 Google AI Studio

訪問 aistudio.google.com/apikey，使用您的 Google 帳號登入。

2

建立 API 金鑰

在控制台中建立新的 API 金鑰，複製並安全保存。請確保金鑰權限設定正確。

3

設定付款方式

在 Google Cloud Console 中設定信用卡付款資訊。這是使用 API 的必要步驟，Google 會根據實際使用量收費。

4

輸入應用程式

在 PhotoContinuum 中輸入您的 API 金鑰，系統會安全儲存在裝置的鑰匙串中。

5

開始創作

設定完成！現在您可以開始享受 AI 照片延續的魅力。首次使用時，Google 提供免費額度供您體驗。

常見問題

為什麼無法生成圖片？

無法生成圖片是因為違反了 Gemini 生圖規定。這個規定讓大家更安全，不會被惡意照片干擾。請確保您的描述內容符合 Google 的使用政策。

使用安全嗎？完全隱私嗎？

完全安全且完全隱私。App 完全隱私，沒有留任何資料。所有照片處理都在您的裝置本地進行，我們無法存取您的任何照片或個人資料。

免費嗎？

目前 Gemini 有免費額度，您只需支付直接向 Google 使用 AI 模型的費用，通常每次處理約 $0.01-0.05 美元。

如何申請 Gemini API 金鑰？

前往 aistudio.google.com/apikey，使用 Google 帳號登入，申請 API 金鑰，並設定信用卡付款。重點是 Google 帳號登入、申請 API 金鑰、信用卡付款。

支援哪些圖片格式？

支援 JPEG、PNG、HEIC 等常見格式。應用程式會自動最佳化圖片尺寸和品質，確保最佳的 AI 處理效果。
